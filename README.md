## Project Title
**Credit Card Fraud Detection**

---

## Project Description
Credit card fraud detection is a critical problem due to the highly imbalanced nature of transaction data.
In this project, machine learning models are used to identify fraudulent transactions by learning patterns
from historical transaction data.

The project includes data preprocessing, exploratory data analysis, feature handling, model training,
and performance evaluation using appropriate metrics for imbalanced datasets.

---

## Objectives
- Understand and handle imbalanced datasets
- Perform data preprocessing and feature analysis
- Build and evaluate machine learning classification models
- Use suitable evaluation metrics such as Precision, Recall, and F1-score
- Gain practical experience with real-world financial data

## Model Evaluation
Due to class imbalance, model performance is evaluated using:
- Precision
- Recall
- F1-score
- Confusion Matrix

These metrics provide a more reliable assessment than accuracy alone.
